**Super Mario Bros Clone**
---
A faithful 2D clone of the original *Super Mario Bros*, developed using Java with the Java Swing Library from scrtach. 

**Controls**
---
| Key       | Action          |
|-----------|-----------------|
| 'W'       | Jump            |
| 'A'       | Move left       |
| 'S'       | Crouch          |
| 'D'       | Move right      |
